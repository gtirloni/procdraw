## Building

Prerequisites:

- Visual Studio 2017 with the "Desktop development with C++" workload

Procdraw is built using the Visual Studio command line tools.

To build Procdraw, open a VS 2017 "Developer Command Prompt" and run:

    > nmake
    > nmake test
