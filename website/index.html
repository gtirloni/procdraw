Procdraw is an experimental live programming environment for making procedurally generated graphics.

It's at an early stage in development and you can find it on GitHub at github.com/simonbates/procdraw.

Memory

  • Numbers
  • Booleans
  • Strings
  • Symbols
  • Lists
  • Dictionaries
  • Native code

Interpreters

  • Lisp-like language
  • Dataflow graph
  • L-systems

External world

  • OpenGL graphics
  • Mouse position
  • MIDI controller input
  • Keyboard input

User interfaces

  • REPL
